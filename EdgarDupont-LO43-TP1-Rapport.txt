Git config:
Permet de configurer ou reconfigurer un fichier, changer son nom, ses valeurs, ajouter des caractéristiques etc...

Git init:
Permet de créer un repository ou d'en réinistaliser un.

Git status:
Permet de montrer l'arbre du fichier en HEAD avec ses différents ancêtres et commit.

Git merge:
Permet de joindre différents fichiers de travail en joignant leurs commit.

Git diff:
Permet de montrer les différences entre deux fichiers dûes à leurs commit.

Git blame:
Permet d'avoir un historique des différents commit avec leur auteur.
